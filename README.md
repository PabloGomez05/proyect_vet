# 🐾 Veterinaria Online

¡Bienvenidos a **Veterinaria Online**!  

Este es un proyecto de **Front-end** desarrollado con **HTML y CSS** para la materia Programación II de TSDS. Representa una interfaz amigable y accesible para una veterinaria donde los usuarios pueden registrar los datos de las mascotas.

🌐 **Deploy en GitHub Pages**  
👉 [https://pablogomez05.github.io/proyect_vet/](https://pablogomez05.github.io/proyect_vet/)

---

## 🎯 Propósito del Proyecto

El propósito principal de este proyecto es aplicar y demostrar conocimientos sólidos en el desarrollo de interfaces de usuario utilizando tecnologías web base (HTML5 y CSS3). Se hace especial énfasis en:
- **Diseño Responsive:** Garantizar que la aplicación sea utilizable tanto en dispositivos móviles como en pantallas de escritorio.
- **Accesibilidad Web:** Cumplir con los estándares básicos (uso de etiquetas semánticas, atributos `alt` en imágenes y contraste de colores adecuado) para que la web sea inclusiva.
- **Maquetación Limpia:** Escribir código estructurado y bien documentado.

---

## ✨ Características

- Diseño web adaptable y limpio (Mobile-First / Responsive).
- Formulario detallado para el ingreso de datos de mascotas.
- Estilo moderno basado en un boceto previo con colores accesibles.
- Maquetado 100% en HTML y CSS puro (sin frameworks).
- Uso de etiquetas HTML5 semánticas.

---

## 📂 Estructura del Proyecto

El repositorio está organizado de la siguiente manera:

```text
proyect_vet/
├── assets/             # Carpeta que contiene las imágenes e iconos (logo, viñetas, iconos de contacto).
├── index.html          # Estructura principal de la página web, incluyendo cabecera, formulario y pie de página.
├── styles.css          # Archivo de estilos principales, organizado con variables y media queries.
└── README.md           # Documentación del proyecto.
```

## 🚀 Cómo clonar y ejecutar el proyecto (Instrucciones)

Para obtener una copia local y hacer funcionar el proyecto, sigue estos pasos:

1. **Clonar el repositorio:**
   Abre tu terminal y ejecuta el siguiente comando:
   ```bash
   git clone https://github.com/pablogomez05/proyect_vet.git
   ```

2. **Acceder a la carpeta del proyecto:**
   ```bash
   cd proyect_vet
   ```

3. **Ejecutar la aplicación:**
   Como es un proyecto estático, no necesitas instalar dependencias ni correr un servidor. Simplemente abre el archivo `index.html` con tu navegador web preferido (haciendo doble clic sobre el archivo o arrastrándolo a la pestaña del navegador).
